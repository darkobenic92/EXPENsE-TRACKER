# Expense Tracker Web App

A **full-featured web application** to track personal income and expenses with **user authentication**.  
Built with **HTML, CSS, JavaScript**, and **Supabase** for backend authentication and database storage. Users can securely log in and manage their own transactions in real time.

---

## Features

- **User Authentication** via Supabase (Sign Up / Log In / Log Out)  
- **Personalized Data:** Each user sees only their own transactions  
- **CRUD Operations:** Add and delete expenses  
- **Real-Time Updates:** Transaction data stored and retrieved from Supabase  
- **Responsive Design:** Works on desktop and mobile devices  
- **User-Friendly UI:** Clean, intuitive interface

---

## Tech Stack

- **Front-End:** HTML5, CSS3, JavaScript (ES6+)  
- **Back-End / Database:** Supabase (authentication and database)  
- **Deployment:** GitHub Pages

---


## How to Use the App

1. Open the **live demo** or `index.html` in your browser.  
2. **Sign Up** or **Log In** with your email and password.  
3. To **add a transaction**:  
   - Enter a description (e.g., "Groceries")  
   - Enter the amount (positive for income, negative for expense)  
   - Click **Add**  
4. To **delete a transaction**, click the **delete button** next to it.  
5. Your **total balance, income, and expenses** are updated automatically.  
6. Log out when finished to keep your data secure.
